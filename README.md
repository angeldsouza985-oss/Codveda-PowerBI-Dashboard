# Stock Market Dashboard using Power BI

This project was completed as part of the Codveda Data Analytics Internship.

## Project Objective
The objective of this project was to create an interactive dashboard for analyzing stock market trends and financial data.

## Dashboard Features
- KPI cards
- Line chart
- Bar chart
- Scatter plot
- Interactive slicer/filter
- Dark-themed dashboard design

## Tools Used
- Power BI
- Stock Prices Dataset

## Output
Successfully built an interactive dashboard for stock market analysis and visualization.
